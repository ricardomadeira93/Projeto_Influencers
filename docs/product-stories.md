# Product stories and personas

## Persona: Low-end creator
- Context: Records tutorials on a budget laptop with webcam overlay in one stream.
- Pain: Cannot afford heavy editing software or manual reframing.
- Story: As a creator, I upload one file and get multiple vertical clips with captions so I can post daily.

## Persona: Dev tutor
- Context: Explains bugs, frameworks, and coding workflows.
- Pain: Long videos hide high-value moments.
- Story: As a dev tutor, I need AI-suggested moments with hooks and reasons so I can pick clips fast.

## Persona: Educator
- Context: Reuses class recordings for social channels.
- Pain: Needs platform-ready metadata and fast republishing.
- Story: As an educator, I want a publish pack and manual platform links so posting stays simple.

## Key scripts
- Hook script: "In the next 30 seconds, here's the exact fix..."
- Outcome script: "By the end of this short, you'll know how to..."
- CTA script: "Comment your niche and I will post part 2."
