# Early user testing documentation

## Test protocol
- Session length: 20 minutes.
- Tasks:
  1) Upload a 10-minute tutorial
  2) Adjust webcam crop
  3) Export one clip and copy metadata pack
  4) Open at least one platform upload page

## Success metrics
- Time-to-first-export under 8 minutes
- 80% of users understand publish pack without guidance
- At least 3 clips rated "post-ready" by user

## Feedback flow in app
- Use the in-app Feedback modal after export.
- Tag issue category manually in follow-up sheet:
  - Upload reliability
  - Clip quality
  - Caption readability
  - Metadata usefulness

## Documentation fields
- User persona
- Device performance tier
- Upload duration and file size
- Completion status
- Top 3 friction points
