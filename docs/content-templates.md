# Content templates

## Title hooks
- Stop this bug in 30 seconds
- The one fix I use every day
- Beginner mistake to avoid today
- Fast tutorial: from zero to output
- Why your approach fails (and what works)

## Description template
Use this formula:
1) Problem line
2) One-paragraph quick fix
3) CTA for follow-up question

Example:
"Bug won't compile? Here's the fastest path to isolate the root cause and patch it. Drop your stack in comments for part 2."
