# Team roles and responsibilities

## Product architect + full-stack engineer (this repo)
- Define scalable MVP architecture (UI/API/worker separation).
- Implement upload, queue lifecycle, worker processing, and cleanup TTL.
- Build dashboard UX for crop config, export clips, and publish pack.
- Prepare phase 1.5 direct publish scaffolding behind feature flags.

## Content/product collaborator scope
- Market research and problem validation.
- Landing page copy and onboarding messaging.
- Beta recruiting operations and communication.
- Sample output curation for creator niches.
- Caption/description template creation.
- Early user testing interview documentation and synthesis.

## Feedback loop ownership
- In-app modal collects raw user feedback.
- Weekly review of feedback + usage logs to prioritize roadmap.
- Track drop-off points from upload to first export.
